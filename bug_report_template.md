# Шаблон баг-репорта

## Краткое описание
Система помечает арест как «выполнен» без подтверждения от банка.

## Шаги воспроизведения
1. Открыть форму наложения ареста
2. Ввести ИНН `770708389309`, выбрать банк Сбербанк
3. В Charles Proxy включить Breakpoint на запросе к банку
4. Отправить запрос → прервать соединение на 15-й секунде
5. Проверить статус операции

## Фактический результат
Статус = «Арест наложен»

## Ожидаемый результат
Статус = «В обработке» до получения ответа от банка

## Приоритет
Блокер (юридические риски)
